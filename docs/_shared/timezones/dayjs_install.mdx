**1.** To install the Day.js library with npm you will need to run the following commands:

```bash
npm install dayjs
```

**2.** After installing the library, for timezones to work properly with the Day.js library, the `utc` and `timezone` plugins need to be included. You'll also need to import them using the dayjsTimezone object from Mobiscroll:

<CodeBlock language="javascript" title="Day.js library import">
import dayjs from 'dayjs';{'\n'}
import timezone from 'dayjs/plugin/timezone';{'\n'}
import utc from 'dayjs/plugin/utc';{'\n'}
import {'\{'} dayjsTimezone {'\}'}  from '@mobiscroll/{props.framework}';{'\n'}
{'\n'}
dayjs.extend(utc);{'\n'}
dayjs.extend(timezone);
</CodeBlock>

**3.** Then set the mobiscroll's reference to the imported library:

<CodeBlock language="javascript" title="Day.js library setup">
// highlight-next-line{'\n'}
dayjsTimezone.dayjs = dayjs;
</CodeBlock>
