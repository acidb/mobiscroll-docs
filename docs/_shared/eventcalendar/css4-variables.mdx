This page lists the Eventcalendar-specific CSS4 variables that can be overridden to customize colors and visual states.
For general theming concepts and shared (generic) theme variables, see the [CSS variables overview](../theming/css4-variables).

:::info
Variable names and available options may differ between themes, so make sure to override the variables that correspond to the active theme.
:::

---

## Eventcalendar variables

Variables can be overridden globally on `:root` or scoped to a specific container.

```scss
:root {
  --mbsc-ios-eventcalendar-accent: #007aff;
  --mbsc-ios-eventcalendar-event-color: #34c759;
}

.my-calendar {
  --mbsc-material-eventcalendar-accent: #6200ee;
}
```

### iOS theme

These variables apply when the `ios` theme is active.


```scss
:root {
  /* Core */
  --mbsc-ios-eventcalendar-accent: ;              /* Selected day, active state, primary highlight color */
  --mbsc-ios-eventcalendar-on-accent: ;           /* Text/icons shown on accent background */

  --mbsc-ios-eventcalendar-bg-color: ;            /* Main calendar background */
  --mbsc-ios-eventcalendar-text-color: ;          /* Default text color (days, labels) */
  --mbsc-ios-eventcalendar-border-color: ;        /* Grid lines, separators */

  --mbsc-ios-eventcalendar-event-color: ;         /* Event background color */
  --mbsc-ios-eventcalendar-event-text: ;          /* Text color inside events */
  --mbsc-ios-eventcalendar-event-mark: ;          /* Small event dot / indicator color */

  --mbsc-ios-eventcalendar-highlight: ;           /* Today / highlighted day background */
  --mbsc-ios-eventcalendar-time-color: ;          /* Time labels, week numbers */

  --mbsc-ios-eventcalendar-disabled-bg-color: ;   /* Disabled day background */
  --mbsc-ios-eventcalendar-parent: ;              /* Background of parent days (outside month) */
  --mbsc-ios-eventcalendar-parent-hover: ;        /* Hover background of parent days */

  --mbsc-ios-eventcalendar-focus: ;               /* Focus outline color (keyboard nav) */
  --mbsc-ios-eventcalendar-success: ;             /* Success state color */
  --mbsc-ios-eventcalendar-error: ;               /* Error state color */

  --mbsc-ios-eventcalendar-popup-bg-color: ;      /* Event popup background */
  --mbsc-ios-eventcalendar-popup-list-bg-color: ; /* Popup event list background */

  /* Calendar view */
  --mbsc-ios-calendar-accent: ;           /* Calendar selection color */
  --mbsc-ios-calendar-on-accent: ;        /* Text on selected day */
  --mbsc-ios-calendar-bg: ;               /* Calendar surface background */
  --mbsc-ios-calendar-popup-bg: ;         /* Popup surface background */
  --mbsc-ios-calendar-text: ;             /* Calendar text */
  --mbsc-ios-calendar-border: ;           /* Calendar grid border */
  --mbsc-ios-calendar-event: ;            /* Event background (calendar view) */
  --mbsc-ios-calendar-event-text: ;       /* Event text (calendar view) */
  --mbsc-ios-calendar-mark: ;             /* Event dots (calendar view) */
  --mbsc-ios-calendar-highlight: ;        /* Highlighted day background */
  --mbsc-ios-calendar-focus: ;            /* Focus outline */
  --mbsc-ios-calendar-week-nr: ;          /* Week number color */

  /* List */
  --mbsc-ios-list-bg-color: ;             /* List background */
  --mbsc-ios-list-bg-color-popup: ;       /* List background inside popup */
  --mbsc-ios-list-text-color: ;           /* List item text */
  --mbsc-ios-list-border-color: ;         /* List separators */
  --mbsc-ios-list-header-bg-color: ;      /* List header background */
  --mbsc-ios-list-header-text-color: ;    /* List header text */
}
```
