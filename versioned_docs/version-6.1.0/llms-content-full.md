> Load only the file matching your framework or domain — unless comparing across frameworks.

This file is an index of framework-specific full documentation files. Each linked
file is self-contained and covers the complete Mobiscroll UI API for that framework,
for both v6 (current) and v5 (legacy). For server-side calendar integration, see
Mobiscroll Connect below. V5 files are optional — use them only if your project is
on Mobiscroll 5.x.

## Framework Documentation (v6 — current)

Each file covers: Eventcalendar, Datepicker, Select, Forms, Popup, Notifications, theming, and core concepts

- [Mobiscroll for JavaScript — Full](llms-javascript-full.txt): Complete JavaScript API reference and guides.
- [Mobiscroll for React — Full](llms-react-full.txt): Complete React API reference and guidess.
- [Mobiscroll for Angular — Full](llms-angular-full.txt): Complete Angular API reference and guides.
- [Mobiscroll for Vue — Full](llms-vue-full.txt): Complete Vue 3 API reference and guides.
- [Mobiscroll for jQuery — Full](llms-jquery-full.txt): Complete jQuery API reference and guides.

## Calendar Integration

- [Mobiscroll Connect — Full](llms-connect-full.txt): Complete Mobiscroll Connect API reference — server-side calendar integration (Google Calendar, Outlook, Apple Calendar, CalDAV), OAuth 2.0, REST API, webhooks, and event sync. Backend/server-side only — not a UI component library.
