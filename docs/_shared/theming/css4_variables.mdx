Mobiscroll components can be styled by overriding CSS4 color variables.
In addition to Sass variables, these CSS variables let you customize colors at runtime without rebuilding styles.

Mobiscroll uses a **two-level CSS variable system**:

- **Theme (generic) variables** define the base color palette for a theme (`ios`, `material`, `windows`)
- **Component variables** are component-specific and inherit their values from the theme variables.

Components never use hardcoded colors.
They always resolve colors from theme variables, making global theming predictable.

Overriding a theme variable updates all dependent components.
Component variables can still be overridden for fine-grained control.

**Using theme color variables**

Theme variables can be overridden in CSS by redefining them on `:root`.

```scss
/* Override the iOS accent color */
:root {
  --mbsc-ios-accent: #ff3b30;
}
```

Here's the complete list of the CSS4 **theme (generic)** color variables, grouped by themes.

---

## iOS color variables

```scss title="Available color variables and defaults for iOS"
:root {
  /* iOS theme – light */
  --mbsc-ios-background;              /* Calendar bg */
  --mbsc-ios-background-alt;          /* Page bg */
  --mbsc-ios-surface-1;               /* Popup bg, disabled bg */
  --mbsc-ios-surface-2;
  --mbsc-ios-surface-3;
  --mbsc-ios-surface-4;               /* Segmented selected bg */
  --mbsc-ios-surface-5;               /* Button bg, switch track, segmented bg */
  --mbsc-ios-surface-6;               /* Timeline parent resource bg */
  --mbsc-ios-surface-7;               /* Form control bg inside popup */
  --mbsc-ios-text;
  --mbsc-ios-text-soft;
  --mbsc-ios-accent;
  --mbsc-ios-accent-soft;             /* Calendar event bg */
  --mbsc-ios-border;

  --mbsc-ios-on-accent-soft;
```

## Material color variables

```scss title="Available color variables and defaults for Material"
:root {
  /* Material theme – light */
  --mbsc-material-accent;                     /* Primary / brand */
  --mbsc-material-background;                 /* Main surface (component backgrounds) */
  --mbsc-material-text;                       /* Main text */
  --mbsc-material-outline;                    /* Outlines (e.g. button/switch outline) */
  --mbsc-material-secondary-container;        /* Secondary container surface */
  --mbsc-material-surface-container-high;     /* Popup / elevated surface */
  --mbsc-material-surface-container-highest;  /* Input box / highest surface */
  --mbsc-material-on-accent;                  /* Text/icons on accent */
  --mbsc-material-on-background;              /* Text/icons on background */
  --mbsc-material-on-surface;                 /* Text/icons on surface */
  --mbsc-material-on-surface-variant;         /* Input/checkbox/radio borders, subtle text */
  --mbsc-material-on-surface-variant;         /* (same name, used as “variant” role) */
  --mbsc-material-accent-soft;                /* Segmented selected bg */
  --mbsc-material-border;                     /* Borders (datepicker, segmented, etc.) */
  --mbsc-material-surface-1;                  /* Stepper disabled button bg */
  --mbsc-material-surface-2;                  /* Timeline resource parent bg */
  --mbsc-material-surface-3;                  /* Timeline disabled bg / select input focused */
  --mbsc-material-surface-4;                  /* Timeline disabled cell bg / text */
  --mbsc-material-surface-5;                  /* Timeline resource parent hover */
  --mbsc-material-action-error;               /* Error state */

  /* Dark variants are also included using the `-dark-` prefixed variables. */
  --mbsc-material-dark-accent;
  --mbsc-material-dark-background;
  --mbsc-material-dark-text;
  --mbsc-material-dark-outline;
  --mbsc-material-dark-secondary-container;
  --mbsc-material-dark-surface-container-high;
  --mbsc-material-dark-surface-container-highest;
  --mbsc-material-dark-on-accent;
  --mbsc-material-dark-on-background;
  --mbsc-material-dark-on-surface;

  --mbsc-material-dark-surface-3;
  --mbsc-material-dark-surface-4;
  --mbsc-material-dark-surface-5;
  --mbsc-material-dark-action-error;
}
```
