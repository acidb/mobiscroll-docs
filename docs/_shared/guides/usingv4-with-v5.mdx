The 5th major version of Mobiscroll contains a [limited number of components](http://help.mobiscroll.com/en/articles/4703511-what-s-the-difference-between-v4-and-v5), compared to v4. 

If you need to keep using some components from v4, which are not present in the newer version, please follow [this guide](http://help.mobiscroll.com/en/articles/4656790-using-mobiscroll-v4-alongside-v5).
