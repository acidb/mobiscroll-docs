This page lists the Form components specific CSS4 variables that can be overridden to customize colors and visual states.
For general theming concepts and shared (generic) theme variables, see the [CSS variables overview](../theming/css4-variables).

:::info
Variable names and available options may differ between themes, so make sure to override the variables that correspond to the active theme.
:::

---

## Form components variables

Variables can be overridden globally on `:root` or scoped to a specific container.

```scss
:root {
  --mbsc-button-standard-bg-color: #6200ee;
  --mbsc-button-standard-text-color: #ffffff;

  --mbsc-button-outline-border-color: #9e9e9e;
  --mbsc-button-outline-text-color: #6200ee;
}

.my-component {
  --mbsc-button-standard-bg-color: #e91e63;
}
```

:::info
**Theming & dark mode**

Variables can be overridden globally on `:root`, which affects all active themes.

To target a specific theme, use its theme selector:

- `.mbsc-windows`
- `.mbsc-windows-dark`
- `.mbsc-ios`
- `.mbsc-ios-dark`
- `.mbsc-material`
- `.mbsc-material-dark`
:::

## Button variables

### iOS theme

These variables apply when the `ios` theme is active.


```scss
:root {
  /* Standard button */
  --mbsc-button-standard-bg-color: ;      /* Default (filled) button background */
  --mbsc-button-standard-text-color: ;    /* Default (filled) button text color */

  /* Outline button */
  --mbsc-button-outline-border-color: ;   /* Outline button border color */
  --mbsc-button-outline-text-color: ;     /* Outline button text color */

  /* Flat button */
  --mbsc-button-flat-text-color: ;        /* Flat button text color */

  /* States */
  --mbsc-button-focus: ;                  /* Focus outline color */
  --mbsc-button-disabled-bg: ;            /* Disabled button background */
  --mbsc-button-disabled-text: ;          /* Disabled button text color */
}

```
### Windows theme

These variables apply when the `windows` theme is active.

```scss
:root {
  /* Standard button */
  --mbsc-button-standard-bg-color: ;      /* Default (filled) button background */
  --mbsc-button-standard-text-color: ;    /* Default (filled) button text color */

  /* Outline button */
  --mbsc-button-outline-border-color: ;   /* Outline button border color */
  --mbsc-button-outline-text-color: ;     /* Outline button text color */

  /* Flat button */
  --mbsc-button-flat-text-color: ;        /* Flat button text color */

  /* Focus */
  --mbsc-button-focus: ;                  /* Focus color */
  --mbsc-button-focus-contrast: ;         /* Contrast color used with focus state */

  /* Disabled */
  --mbsc-button-disabled-bg: ;            /* Disabled button background */
  --mbsc-button-disabled-text: ;          /* Disabled button text color */
}
```
### Material theme

These variables apply when the `material` theme is active.

```scss
:root {
  /* Standard (filled) button */
  --mbsc-button-standard-bg-color: ;      /* Filled button background (accent fallback) */
  --mbsc-button-standard-text-color: ;    /* Filled button text color (on-accent fallback) */

  /* Outline button */
  --mbsc-button-outline-border-color: ;   /* Outline button border color (outline fallback) */
  --mbsc-button-outline-text-color: ;     /* Outline button text color (accent fallback) */

  /* Flat button */
  --mbsc-button-flat-text-color: ;        /* Flat button text color (accent fallback) */

  /* Focus */
  --mbsc-button-focus: ;                  /* Focus outline / highlight color (accent fallback) */

  /* Disabled */
  --mbsc-button-disabled-bg: ;            /* Disabled button background (text fallback) */
  --mbsc-button-disabled-text: ;          /* Disabled button text color (text fallback) */
}
```

## Switch variables

### iOS theme

These variables apply when the `ios` theme is active.


```scss
:root {
  /* Default state */
  --mbsc-switch-handle-color: ;             /* Switch handle (thumb) color (surface-2 fallback) */
  --mbsc-switch-track-color: ;              /* Switch track color (surface-5 fallback) */

  /* Selected state */
  --mbsc-switch-selected-handle-color: ;    /* Handle color when checked (background fallback) */
  --mbsc-switch-selected-track-color: ;     /* Track color when checked (action-success fallback) */

  /* Focus */
  --mbsc-switch-focus: ;                    /* Focus outline color */
}

```
### Windows theme

These variables apply when the `windows` theme is active.

```scss
:root {
  /* Default state */
  --mbsc-switch-handle-color: ;             /* Switch handle (thumb) color (strong border fallback) */
  --mbsc-switch-track-color: ;              /* Switch track background (none fallback) */
  --mbsc-switch-border: ;                   /* Switch border color (strong border fallback) */
  --mbsc-switch-text: ;                     /* Switch text color */

  /* Selected state */
  --mbsc-switch-selected-handle-color: ;    /* Handle color when checked (background fallback) */
  --mbsc-switch-selected-track-color: ;     /* Track color when checked (accent fallback) */

  /* Interaction */
  --mbsc-switch-outline: ;                  /* Outline color */
  --mbsc-switch-hover-color: ;              /* Hover background color */
  --mbsc-switch-focus: ;                    /* Focus color (may be unused) */
}

```
### Material theme

These variables apply when the `material` theme is active.

```scss
:root {
  /* Default state */
  --mbsc-switch-handle-color: ;             /* Switch handle (thumb) color (outline fallback) */
  --mbsc-switch-track-color: ;              /* Switch track background (surface-container-highest fallback) */
  --mbsc-switch-border-color: ;             /* Switch border color (outline fallback) */

  /* Selected state */
  --mbsc-switch-selected-handle-color: ;    /* Handle color when checked (on-accent fallback) */
  --mbsc-switch-selected-track-color: ;     /* Track color when checked (accent fallback) */
}
```
