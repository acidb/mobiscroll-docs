**1.** To install the moment-timezone library with npm you will need to run the following commands:

```bash
npm install moment-timezone
```

**2.** After the library is installed, you will have to import it with the `momentTimezone` object from mobiscroll:

<CodeBlock language="javascript" title="Moment library import">
import moment from 'moment-timezone';{'\n'}
import {'\{'} momentTimezone {'\}'} from '@mobiscroll/{props.framework}';
</CodeBlock>

**3.** Then set the mobiscroll's reference to the imported library:

<CodeBlock language="javascript" title="Moment library setup">
import moment from 'moment-timezone';{'\n'}
import {'\{'} momentTimezone {'\}'} from '@mobiscroll/{props.framework}';{'\n'}
// highlight-next-line{'\n'}
momentTimezone.moment = moment;
</CodeBlock>
